# 地球收藏夹：Pro 与盈利运营计划

## 一、先卖一个清楚的产品

- 免费版：保留 3D 地球、基础收藏、路线、日记和数据备份。
- Pro：99 元/年，提供跨设备云同步、更多照片容量、高清无水印手帐、更多共享空间和路线样式。
- 创始会员：前 100 名 59 元/年，用来验证是否真的有人愿意付费。
- 双人/家庭版：199 元/年，支持 2–5 人共同记录。

核心原则：不把用户已经收藏的回忆锁起来。收费的是“保存得更久、制作得更漂亮、分享得更方便”。

## 二、给朋友免费 Pro 名额

创建者账号预留 5 个永久 Pro 赠送名额，专门给朋友测试，不公开售卖。

正式上线后，普通 Pro 会员每年获得 3 个“30 天好友体验码”。朋友领取后必须登录自己的账号，Pro 权益绑定账号，不绑定某台手机。

不要只使用 `localStorage` 保存 Pro 状态，否则用户换 Safari、微信或手机后会失效。后期在 CloudBase 新建 `earth_user_entitlements` 集合，每个账号保存：

- `ownerId`：账号 ID
- `plan`：free、pro 或 family
- `source`：purchase、admin-gift 或 invite
- `startsAt` 与 `expiresAt`：开始和到期时间
- `status`：active、expired 或 revoked
- `note`：例如“创建者好友名额 1/5”

付款、生成体验码和发放永久名额必须由云函数或管理后台执行；网页只读取结果，不能让浏览器自己把账号改成 Pro。

## 三、最适合现在的赚钱顺序

1. 注册 Involve Asia。
2. 申请 Klook Affiliate。
3. 注册 Travelpayouts，只选择与内容直接相关的酒店、门票或旅行服务。
4. 网站增加 3–5 篇公开内容和真实用户示例。
5. 有稳定访问量后再申请 AdSense。
6. 用户数据和案例足够后，直接联系 eSIM、留学服务和照片打印品牌。

联盟链接必须放在真正有帮助的内容里，例如路线下面推荐对应门票、旅行日志下面推荐同城体验。不要在保存私人回忆的页面插入广告。

## 四、先写的 5 篇公开内容

1. 《如何把一趟旅行做成 3D 地球足迹》
2. 《加州旅行路线示例：旧金山到洛杉矶》
3. 《留学生的一周：照片、打卡和心情怎么整理》
4. 《微信、Safari 和 Chrome 如何同步同一个地球收藏夹》
5. 《8 张照片自动做成旅行手帐的完整示例》

每篇内容都要有真实截图、实际路线或手帐成品，再自然加入 1–2 个相关服务链接。

## 五、上线顺序

1. 先上线 Pro 介绍页和创始会员意向登记。
2. 建立账号权益集合和管理员赠送功能。
3. 给 5 位朋友发永久 Pro，连续测试不同手机和浏览器。
4. 接入付款，并明确价格、有效期、退款方式和是否自动续费。
5. 再开放 30 天好友体验码。
6. 最后逐步增加联盟内容、实体照片书和品牌合作。
