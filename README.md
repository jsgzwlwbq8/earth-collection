# 地球收藏夹

一个使用 HTML、CSS、原生 JavaScript 和 Globe.gl 制作的手机网页应用。

用户可以选择国家和城市，上传照片，并把建筑、美食、风景、动物、物品或其他回忆收藏到 3D 地球上。数据和照片使用 IndexedDB 保存在当前浏览器中。

第二版增加了自动定位、地址搜索、OpenStreetMap 平面地图、更多国家和加州城市，以及按月份查看照片与心情的回忆日历。

日记入口使用日记本图标，日记页可以直接添加照片日记；照片添加区会一直显示选择或更换照片按钮。3D 地球使用更高像素密度和更细的球面网格显示。

中国城市选择列表包含全国地级行政区和县级市；台湾城市的所属地区统一显示为“中国台湾”。城市名单保存在 `china-cities.js`，选择城市后会按需查询坐标并转动地球。

美国、日本、韩国、新加坡和欧洲主要旅行国家的热门城市保存在 `popular-cities.js`。城市选项会同时提示代表景点；具体景点、街道或地址仍可使用 OpenStreetMap 搜索。

足迹路线会按照收藏照片的日期连接不同地点。地球页面左上角有“我的足迹路线”入口；路线页中每一段都可以直接选择“飞机”或“高铁”。飞机路线显示较高的蓝色飞行弧线，高铁路线显示贴近地球的金色轨迹。路线页可以让最后一站自动连回出发点。

旅行日志可以按日期汇总照片、地点、心情和文字记录，也可以补充日志标题与旅行中最好玩的事情。支持的手机会打开系统分享菜单并尽量附带前六张照片；其他浏览器会复制整理好的分享文字。

足迹路线支持飞机、高铁、大巴和邮轮，每一段路线都可以单独修改交通方式，地球会用不同颜色和高度显示。

留学日记入口使用 🎓 图标。它包含留学生等级、经验条、六个支线任务、每日小故事和照片章节；添加收藏时勾选“收录到留学日记”，照片就会同时出现在地球和留学日记中。留学日记内容会一起包含在导出备份里。

留学日记增加每日风景与美食打卡，每种每天可以完成一次，每次增加 1 XP。累计打卡 6、66、100、200、520、666、888 天时会自动解锁阶段徽章；打卡和徽章进度会包含在导出备份中。

顶部的 👥 入口是多人共享日志。连接 CloudBase 后，可以创建共享空间、复制邀请码、加入朋友的空间、发布照片和心情，并给对方的照片留言。具体云端设置请阅读 `CLOUDBASE-共享日志设置.md`。

共享日志里也提供不需要 CloudBase 的“朋友共享包”。一个 `.html` 文件既是可以直接打开浏览的只读日记网页，也能由另一台设备导入并自动合并。合并时保留本机内容，只新增缺少的记录，并用更新时间较新的同一条记录更新旧版本；留学日记、打卡进度和朋友留言也会一起合并。

右上角原来的导入、导出箭头已合并为“💾 备份与恢复”。第一次点击“开始收藏”会显示网站用途和主页按钮说明；勾选“下次不再自动提醒”后不会再次自动弹出，仍可点击右上角“?”重新查看。

首页增加流星许愿、小火箭和火星轨道装饰。火星位置会依据 NASA/JPL 公布的行星轨道参数和当天日期进行近似计算，并以屏幕外圈位置示意；它不是用于夜间观测的精密星图。

## 本地预览

在项目文件夹中运行：

```bash
python3 -m http.server 4173
```

然后打开 `http://127.0.0.1:4174/?v=19`。

## 数据说明

- 不需要 npm、后端或数据库账号。
- 不同浏览器的数据不会自动同步。
- 请定期使用右上角的“导出备份”。
- 可以把全部文件直接发布到 GitHub Pages。
